# Sales-Management
A 'Sales Management' application written in python for purchasing items which could:
1.  display items
2.  add items
3.  delete items
4.  update items price
5.  update items quantity
6.  search items
7.  sort by quantity
8.  sort by price 
9.  buy items
10. display daily sales
11. display monthly sales
12. display yearly sales
